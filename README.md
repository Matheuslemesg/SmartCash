# SmartCash

## 📌 Sobre o Projeto
O **SmartCash** é um sistema de gestão financeira desenvolvido em C# para ajudar usuários a monitorar suas receitas, despesas e balanços financeiros de forma eficiente. 

## 🚀 Funcionalidades
- Cadastro de receitas e despesas
- Categorização de transações financeiras
- Geração de relatórios financeiros
- Dashboard interativo para visualização de saldo
- Controle de metas financeiras

## 🛠️ Tecnologias Utilizadas
- **Back-end:** C# (.NET)
- **Banco de Dados:** SQL Server
- **Front-end:** Vue.js
- **Ferramentas:** Swagger para API

## 📦 Como Instalar e Rodar o Projeto
### 1️⃣ Clonar o repositório:
```sh
git clone https://github.com/Matheuslemesg/SmartCash
cd smartcash
```

### 2️⃣ Configurar o ambiente:
- Instale o **.NET SDK** (se ainda não tiver instalado)
- Configure a string de conexão com o banco de dados em `appsettings.json`

### 3️⃣ Rodar a aplicação:
```sh
dotnet run
```

Acesse o Swagger para testar a API:
```
http://localhost:5151/swagger/index.html
```
